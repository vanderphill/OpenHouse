Step One

Change Admin Password

The default password for the admin account is “admin”. It is imperative that you change your password to something more secure. To change the admin password click here (you can also navigate to the “change password” in the admin menu).
Enter the old password and your new password. Do not lose or forget your password.

Step Two

Create Warehouse Space

There are two methods fro creating warehouse space. the first to create individual locations, the second is to create racks of locations. both of these methods can be found in the location management section in the Admin Menu

Creating Locations

locations can be used for both storage and shipping. for example for a special type of product that is tood big for a rack or can not go on a rack for some reason you can make a special location for it(i.e. a silo, a refridgerator, a secured location or an outdoor location). it should also be used for shipping staaging locations. i.e. truck1, truck2, UPS, ect.

example warehouse layout




Creating Racks

when creating racks you will be asked for a rack number, the number of columns and the number of rows. racks may have upto 99 columns and upto 26 rows and you may create up to 99 racks. that is a possible 254826 locations. rack location numbers are composed of the rack number, column number and row height indicated with a letter starting with 'A' (see example below)




Removing Locations

both rack and non racks locations can be removed. make sure the loction is empty before deleting as this may result in the loss of inventory data. see example below where location 0103C has been removed.





Step Three

Add Users

To add users navigate to user management in the Admin Menu and click on 'Add User' it will aask for a username and password. make sure only the user knows the password.

Remove Users

In the event an account needs to be disabled, navigate to remove user in usermanagement and select the user from the drop down menu and click remove. in the event that a password is lost or forgoten the admin will have to delete the users account and then re-create it.
